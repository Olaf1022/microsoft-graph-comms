Graph Core SDK and Graph Calling SDK Concepts

This article describes the key concepts in order to effectively utilize the Graph Core SDK and the Graph Calling SDK.

Graph Core SDK

The DefaultContainerClient object is the entry point to the Graph Core SDK. The majority of the Graph Core SDK is automatically generated by the Graph SDK generator using the OData resource model. It reflects the REST-ful wire protocol provided by the back-end service.

Graph Calling SDK and Stateful Client Builder

The @Microsoft.Graph.Calls.IStatefulClient object is the entry point to the Graph Calling SDK. This SDK is designed for stateful services. It provides additional support on top of the Graph Calling Core SDK for resource state and media management. The @Microsoft.Graph.Calls.StatefulClientBuilder is the object used to construct a new IStatefulClient with the desired settings.

For more details on how to configure your bot please review the Setting up stateful bots with the Graph Calling SDK article.
